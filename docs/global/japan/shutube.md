# 内田秀のVlog

<p><Badge type="info" text="🌱 Seedlings" /></P>

## 高尾山
- 男坂：陡坡
- 女坂：緩坡
- 六根清淨石車：可開運，共18座
- 要爬約兩小時
- 離高尾山口站最近

## 代代木公園
- 最大的東京都心公園
- 曾是代代木之原和華盛頓高地
- 在明治神宮旁，離原宿站最近

## 由比濱
- 屬於湘南海岸，面朝相模灣
- 和鎌倉海濱公園、極樂寺相距不遠
- 有出現在《海街日記》中
- 離由比濱站最近

## 昭和紀念公園
- 適合賞櫻和賞銀杏，要門票
- 離西立川站最近
